# 3д-модель брелка
Реализованного в виде логотипа it-college

## Обоснование текущего состояния файлов

В проекте сейчас присутствует заметное количество временных и промежуточных файлов. Это нормальная ситуация, связанная с активным этапом моделирования брелка: мы одновременно проверяем масштабирование, корректность геометрии, совмещение слоёв, тестовые варианты экспорта и разные способы сборки модели.

В процессе работы создаются черновые .blend, .stl, .obj и версии однотипных файлов — это неизбежный побочный эффект, пока финальная модель ещё формируется.

После завершения всех геометрических правок и окончательной сборки будет проведена полная чистка: останутся  итоговые чистые файлы модели и минимально необходимый набор рабочих исходников. Всё остальное будет удалено как лишний технический мусор, накопившийся в ходе активного изучение и разработки.

## **Общие размеры модели (чёрная база + белая накладка)**

* **Высота (Y): 40 мм**
  Это основной габарит, под который масштабирована вся модель.

* **Ширина (X): 35.8 мм**
  Это пропорциональная ширина лого после корректного масштабирования под 40 мм по Y.

* **Полная толщина модели (Z): 3.0 мм**
  Складывается из:

  * чёрная база — **2.2 мм**
  * белая накладка — **0.8 мм**

---

## **Чёрная база (нижний слой)**

* **Размер по X:** 35.8 мм
* **Размер по Y:** 40 мм
* **Толщина по Z:** 2.2 мм
* **Позиция:**

  * Location X = 0
  * Location Y = 0
  * Location Z = 0 (нижняя плоскость модели)

База является опорной частью брелка и задаёт общий масштаб.

---

## **Белая накладка (верхний слой)**

* **Размер по X:** ~33.5 мм
* **Размер по Y:** ~34.2 мм
* **Толщина по Z:** 0.8 мм
* **Позиция для корректного совмещения:**

  * Location X = 0
  * Location Y = 0
  * Location Z = **0.0022 м** (2.2 мм) — ровно на поверхности базы

Белая накладка центрируется по тем же координатам, что и чёрная база, и лежит сверху без зазоров.
